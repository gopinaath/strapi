# Excalidraw Architecture Guide - Simplified Version

## Core Components (Top to Bottom)

### 1. Users (Top)
- Admin Users
- API Users  
- Web Visitors

### 2. Edge Layer
- **CloudFront CDN** → Global distribution
- **WAF** → Security shield

### 3. Regional Infrastructure (Draw a box)

#### Load Balancer
- **ALB** → Distributes traffic
- **WAF** → Admin protection

#### Application Layer (3 boxes for AZs)
- **ECS Tasks** → Strapi containers (2-3 instances)

#### Database Layer
- **RDS Primary** → Main database
- **RDS Standby** → Backup (with sync arrow)

### 4. Storage & Services (Outside main box)
- **S3** → Media files
- **Secrets** → Passwords
- **ECR** → Docker images

## Key Connections

### Traffic Flow
1. Users → CloudFront → ALB → ECS
2. CloudFront → S3 (media)

### App Connections
- ECS → RDS (data)
- ECS → S3 (uploads)
- ECS → Secrets (passwords)

## Important Labels
- "Multi-AZ" on the infrastructure
- "Auto-scaling" on ECS
- "Encrypted" on storage
- "99.9% uptime" as title

## Colors (Simple)
- Orange: AWS services
- Blue: Network/CDN
- Green: Database
